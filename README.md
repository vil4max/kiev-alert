# KievAlert

SwiftUI demo for Ukraine air-raid alert status by region (selected via location), with a map of regional statuses — portfolio piece toward a glanceable CarPlay ALARM/QUIET experience. Demo project; data availability and accuracy are not guaranteed.

## Stack

iOS 18.5+ · Xcode · Swift · SwiftUI · SPM (Domain, Data, Status, Map) · URLSession · MapKit · CoreLocation · String Catalogs (en/ru/uk) · Swift Testing · CarPlay (demo)
